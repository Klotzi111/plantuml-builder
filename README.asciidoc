= Plantuml builder
image:https://travis-ci.org/jboz/plantuml-builder.svg?branch=master["Build Status", link="https://travis-ci.org/jboz/plantuml-builder"]

Tools to generate plantuml diagram from java sources.

== class diagram

This code :
[source,java]
----
include::src/test/java/ch/ifocusit/plantuml/classdiagram/ClassDiagramBuilderTest.java[tags=createSimple]
----

Will get :
[plantuml, class-diagram, png]
plantuml::src/test/resources/class-diagram.plantuml[format=png, alt="Class diagram"]

For more methods see **ClassDiagramBuilderTest** or **PlantUmlBuilderTest** classes
